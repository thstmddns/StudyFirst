2. 간단한 369게임 (1926)

```python
n = int(input())
answer = ''
for num in map(str, range(1, n+1)):
    clap = 0  # 박수친 횟수
    for x in '369':  # 3, 6, 9
        clap += num.count(x)  # 숫자의 3, 6, 9 수만큼 박수 횟수 누적
    if clap:  # 박수 쳤으면 해당 횟수 만큼 '-' 붙임
        answer += ('-' * clap) + ' '
    else:  # 안 쳤으면 현재 숫자 붙임
        answer += num + ' '
print(answer)
```
