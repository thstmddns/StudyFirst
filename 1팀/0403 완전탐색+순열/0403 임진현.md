#### 부분수열의 합

```python
n, s = map(int, input().split())
line = list(map(int, input().split()))
cnt = 0

for i in range(1, (1<<n)): # 크기가 양수인 부분집합이므로, 공집합은 제외
    case = 0
    for j in range(n):
        if i & (1<<j):
            case += line[j]
    if case == s:  # 부분집합의 합이  s 와 같으면 카운트 하기
        cnt += 1

print(cnt)
```
