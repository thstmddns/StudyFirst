### 문자열 비교

``` python
T = int(input())

for i in range(1, T + 1):
    str1 = input()
    str2 = input()
    if str1 in str2 : 
        print(f'#{i} 1') #출력 방법 다르게 하는 것 질문
    else : 
    	print(f'#{i} 0') #출력 방법 다르게 하는 것 질문
```

### 글자수

```python
T = int(input())

for i in range(1, T + 1):
  str1 = input()
  str2 = input()
  dict_a = {}
  for a in str1():
    if a in str2():
      dic_value = len(str2.count(a))
      dict_a[a] = dic_value
  dic_max = max(dict_a.values())
print(f'#{i} {dic_max}') 
```
