### 1289 메모리 복구하기

```python
T = int(input())
for tc in range(1, T+1) :
    N = list(input())
    num = 0
    find = ['0'] * len(N)
    for i in range(len(N)) :
        if N[i] != find[i] :
            for j in range(i, len(N)) :
                find[j] = N[i]
            num += 1
    print(f'#{tc} {num}')
```

### 2805 농작물 수확하기

```python
T = int(input())
for tc in range(1, T+1):
    N = int(input())
    arr = [list(map(int, input())) for _ in range(N)]
    mid = N//2
    start = end = mid
    result = 0
    for i in range(N):
        for j in range(start, end+1):
            result += result[i][j]
        if i < mid :
            start = start -1
            end = end -1
        else :
            start = start +1
            end = end +1
    print(f'#{tc} {result}')
```



### 백준 1244 스위치 켜고 끄기

```
N = int(input())
arr = list(map(int, input().split()))
S = int(input())

for tc in range(S):
    student, switch = map(int, input().split())
    if student == 1:
        for i in range(1, (len(arr) // switch) +1)
            if arr[(switch * i) -1] == 0:
                arr[(switch * i) - 1] =1
            else:
                arr[(switch * i) - 1] = 0

    if student == 2:
        if arr[(switch - 1)] == 0:
            arr[(switch -1)] = 1
        else:
            arr[(switch -1)] =0
```
