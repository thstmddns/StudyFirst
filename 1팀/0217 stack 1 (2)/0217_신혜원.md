## 반복문자

```python
T = int(input())

for tc in range(1, T+1):
    arr = list(input())
    N = len(arr)
    stack = []
    for i in range(N):
        if not stack or stack[-1] != arr[i]:
            stack.append(arr[i])
        elif stack and stack[-1] == arr[i]:
            stack.pop()
    print(f'#{tc} {len(stack)}')
```
