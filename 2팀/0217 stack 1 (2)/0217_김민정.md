반복문자 지우기

```python
T = int(input())
for tc in range(1, T + 1):
    arr = list(input())
    b = []
    while arr:
        # stack이 비어있으면 넣기
        if not b:
            b.append(arr.pop())
        # stack이 비어있지 않을 때
        else:
            a = arr.pop()
            # arr[-1]이 stack의 top과 같지 않으면 stack에 추가
            if a != b[-1]:
                b.append(a)
            # arr[-1]이 stack의 top과 같으면 stack의 top 제거
            else:
                b.pop()
    print(f'#{tc} {len(b)}')
```
