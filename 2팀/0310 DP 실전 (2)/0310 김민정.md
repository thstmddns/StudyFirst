## 11052. 카드 구매하기

```python
n = int(input())
p = [0] + list(map(int, input().split()))
f = [p[0], p[1]]
for i in range(2, n+1):
    max_value = f[0]
    for j in range(1, i+1):
        if f[i-j]+p[j] > max_value:
            max_value = f[i-j]+p[j]
    f.append(max_value)
print(f[n])
```
