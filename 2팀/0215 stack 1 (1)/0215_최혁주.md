# 돌아가는 순서는 알겠지만 말로 설명은 못하겠는... 친구에게 도움을 좀 받은 코드입니다.

w = int(input()) #채워야 하는 직사각형 밑변 너비
cnt = 0 #방법 수
def put(remain, now):
    if remain == 0 and now == w: #꽉 찼다
        global cnt
        cnt += 1
        return #되돌아감

    put(remain - 10, now + 10) #기본 넣는 방식

        put(remain - 20, now + 20)
    return
