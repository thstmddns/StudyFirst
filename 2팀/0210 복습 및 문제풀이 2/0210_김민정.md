#### 1986. 지그재그 숫자

```python
T = int(input())
for tc in range(1, T+1):
    N = int(input())
    result = 0
    for i in range(1, N+1):
        # 홀수면 더하기
        if i % 2:
            result += i
        # 짝수면 빼기
        else:
            result -= i
    print(f'#{tc} {result}')
```
