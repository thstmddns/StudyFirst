### 공지

```
1. 각자의 코드는 스터디 시작 1시간 전까지는 업로드 되어야 합니다.

2. 스터디 전에 다른 팀원의 코드를 읽고, 접근방식을 훑어 본 후 스터디에 참여합니다.

3. 궁금하거나, 이해 안되는 점은 즉시 바로 이야기 해야 합니다.
   그 순간엔 힘들어도 E의 성향으로 변해주세요!
   
4. 多多益善! 다양한 접근방식과 풀이를 보면 도움됩니다. 팀 간에 꼭 코드를 열람하고, 지식을 공유해주세요!

   손해입니다. 화이팅 해봅시다!
```
